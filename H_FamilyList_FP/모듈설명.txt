1. treeview_handlers.py:
목적: 이 모듈은 TreeView 위젯과 관련된 사용자 상호 작용 및 이벤트 중심 동작을 처리하는 역할을 담당합니다.
책임:
마우스 오른쪽 버튼 클릭, 더블클릭, 드래그 앤 드롭 작업 처리와 같은 TreeView에 대한 이벤트 바인딩을 관리합니다.
상황에 맞는 메뉴 표시, 편집 시작, 선택 수행과 같은 사용자 작업에 응답하는 논리를 구현합니다.
일반적으로 UI 컨텍스트에서 이벤트를 처리하고 이러한 이벤트에 대한 응답으로 적절한 작업을 실행합니다.
MVC의 역할: 이 모듈은 View 레이어의 일부로, TreeView가 사용자와 상호 작용하는 방식에 초점을 맞추고 UI가 이러한 상호 작용에 적절하게 응답하는지 확인합니다.
예제 기능:

드래그 앤 드롭 작업 처리.
사용자가 항목을 마우스 오른쪽 버튼으로 클릭하면 상황에 맞는 메뉴가 표시됩니다.
사용자가 상황에 맞는 메뉴에서 "편집" 옵션을 선택하면 편집 모드가 시작됩니다.

2. treeview_utils.py:
목적: 이 모듈은 TreeView 위젯에 대한 유틸리티 기능 및 지원 작업을 제공합니다. TreeView를 조작하는 데 필요한 범용 함수이지만 특정 사용자 이벤트와 연결되지는 않습니다.
책임:
TreeView 데이터(예: JSON 파일)를 로드하고 저장하는 기능.
TreeView 내 노드 확장/축소와 같은 작업을 검색하고, 채우고, 수행하기 위한 유틸리티 기능입니다.
'TreeView'를 관리하기 위해 애플리케이션의 여러 부분에서 사용할 수 있는 범용 메서드입니다.
MVC의 역할: 이 모듈은 View 레이어 내에서 도우미 역할을 합니다. TreeView 관리를 돕는 필수 기능을 제공하지만 사용자 상호작용을 직접 처리하지는 않습니다.
예제 기능:

JSON 파일에서 데이터를 로드하고 TreeView를 채웁니다.
특정 항목에 대해서는 TreeView를 통해 검색합니다.
TreeView에서 특정 수준까지 노드를 확장하거나 축소합니다.

3. tree_utils.py:
목적: 이 모듈에는 좀 더 범용하고 'TreeView'뿐만 아니라 애플리케이션의 다양한 부분에서 사용할 수 있는 하위 수준 유틸리티 함수가 포함되어 있습니다. UI 관련 TreeView보다는 트리와 관련된 핵심 로직에 더 중점을 둡니다.
책임:
레벨 계산, 항목 번호 다시 매기기 또는 레벨에 따른 스타일 결정과 같은 핵심 트리 작업을 처리하는 기능입니다.
이 함수는 다양한 컨텍스트에서 재사용이 가능하며 TreeView의 특정 구현에 묶여 있지 않습니다.
'TreeView' 위젯과 무관하지만 일반적으로 트리형 데이터 구조와 관련된 일반 유틸리티 함수입니다.
MVC의 역할: 이 모듈은 모델 및 뷰 레이어를 모두 지원하는 공유 유틸리티로 볼 수 있습니다. 이는 해당 기능이 필요한 애플리케이션의 어느 곳에서나 사용할 수 있는 트리 구조에 대한 기본 작업을 제공합니다.
예제 기능:

트리 계층 구조에서 항목의 수준을 계산합니다.
위치에 따라 트리의 항목 번호를 다시 매깁니다.
레벨에 따라 항목에 적합한 태그나 스타일을 결정합니다.
주요 차이점:
treeview_handlers.py는 이벤트 중심이며 TreeView가 사용자 작업에 응답하는 방식에 중점을 둡니다. 그것은 사용자가 TreeView와 상호작용할 때 일어나는 일에 관한 것입니다.

treeview_utils.py는 TreeView에 대한 다양한 작업을 지원하는 도우미 기능을 제공합니다. 데이터 로드나 항목 검색과 같은 TreeView에서 특정 작업을 수행하는 방법에 대한 자세한 내용입니다.

tree_utils.py에는 TreeView 위젯에만 국한되지 않지만 애플리케이션의 모든 트리형 데이터 구조와 관련된 핵심 트리 관련 유틸리티가 포함되어 있습니다. '모델' 및 '뷰' 레이어 모두에서 사용할 수 있는 일반 트리 작업에 중점을 둡니다.

개념 요약:
treeview_handlers.py: TreeView의 사용자 상호작용 및 이벤트 처리에 중점을 둡니다.
treeview_utils.py: TreeView 위젯과 관련된 도우미 기능을 제공합니다.
tree_utils.py: 일반적인 의미에서 트리 구조 작업을 위한 핵심 유틸리티 기능을 제공합니다.
